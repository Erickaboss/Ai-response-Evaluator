# AI Response Evaluator

A Python tool that simulates the work of AI training engineers by evaluating the quality of responses generated by AI models like Claude or ChatGPT.

---

## Project Purpose

AI training engineers spend significant time reviewing model outputs and scoring them across multiple quality dimensions. This project automates that process using heuristic-based scoring, making it easy to benchmark and compare responses to the same prompt.

---

## Evaluation Criteria

| Criterion | Weight | Description |
|---|---|---|
| Clarity | 20% | Are sentences concise and easy to understand? |
| Completeness | 25% | Does the response address all key points in the prompt? |
| Reasoning | 25% | Does the response explain the *why* or *how*? |
| Example Usage | 15% | Are concrete examples or code snippets included? |
| Length & Readability | 15% | Is the response an appropriate length with good structure? |

Each criterion is scored 1–10. A weighted final score is computed and a verdict is given.

## How to Run

**1. Clone the repo**
```bash
git clone https://github.com/Erickaboss/Ai-response-Evaluator.git
cd Ai-response-Evaluator/ai_response_evaluator
```

**2. Install Streamlit (for browser UI)**
```bash
pip install streamlit
```

**3. Run in the browser**
```bash
python -m streamlit run app.py
```
Then open http://localhost:8501 in your browser.

**Or run in the terminal (no dependencies)**
```bash
python main.py
```

**Run built-in examples in terminal:**
```bash
python main.py --examples
```

## Notes

- Scoring is heuristic-based and designed for demonstration purposes.
- Criteria weights can be adjusted in `feedback.py` under `CRITERIA_WEIGHTS`.
- The evaluator works best on English-language responses.
